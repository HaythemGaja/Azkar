# Azkar
Azkar (حصن المسلم) Web App
A modern, beautifully designed, and highly interactive Islamic web application tailored for reading daily remembrances (Azkar). It is fully responsive, behaving like a native mobile app on Android/iOS while remaining perfectly usable on a desktop PC.

Key Features:

Three Main Categories: Easily switch between Morning (الصباح), Evening (المساء), and Sleep (النوم) Azkar using a modern fixed bottom navigation bar.

Interactive Counting: Tap the main button to count down repetitions. It features a satisfying visual "pulse" animation. It also includes options to skip an item or undo a completion.

Smart Progress Tracking: A visual progress bar at the top tracks your daily completion. Your progress is saved automatically for the day so you can close the app and return later.

Customization & Accessibility:

Dark & Light Mode: A built-in theme toggle to reduce eye strain depending on the time of day.

Dynamic Font Sizing: Fixed A+ and A- buttons in the top bar allow users to instantly adjust the Arabic text size for optimal readability.

Custom Sorting: A "Reorder" (ترتيب) mode lets users permanently change the sequence of the Azkar to match their personal reading habits.

Real-time Search: A hidden search panel drops down from the top bar, allowing users to instantly filter Azkar by text, virtue (فضل), or reference source.

Native App Feel: Utilizes fixed top and bottom toolbars, smooth scrolling, modern SVG icons, and a layout that respects mobile "safe areas" to provide a premium, app-like experience directly in the browser.
